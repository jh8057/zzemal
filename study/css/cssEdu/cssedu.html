css 문법

                    selector {
                        property : value
                    }
                    
                    ex) * : {
                        color : green;
                    } 
                    - 모든 영역의 font color가 green이 된다.

                    a[href="naver.com"]{
                        color:purple
                    }
                    - a태그 속성중에 href를 가지고있고, href가 "naver.com"인 경우만 color가 purple이 된다.

                    
                    a[href $=".com"]{
                        color:purple
                    } - ".com"으로 끝나는것

FlexBox

float = 이미지를 문단 내 어디에 위치시킬지 정하는 문법이였다.
flexbox에는 container에 지정하는 속성, item에 지정하는 속성이 있다.
main axis , cross axis 중심축 , 수평축이 있다.
div.flexcontainer>div.item.item${$}*8
위 처럼 작성하면 한번에 쉽게 작성할 수 있다.
flex-direction을 통해 방향을 설정하여 배열을 바꿔줄수 있다.
flex-wrap은 window창이 줄어들면 거기에 맞춰 overflow가 일어날 경우 다음 줄로 자동 옮겨주는 기능이다
flex-flow는 flex-direction과 flex-wrap을 한번에 적는게 가능하다.
justify-content은 중심축(main axis)을 기준으로 내용물을 정렬하는 기능이다.
flex-end,space-around 등 다양하다.
align-item 는 교차축(cross axis)을 기준으로 item을 정해진 contain안에서 정렬하는 것이다.
align-content 는 교차축(cross axis)을 기준으로 내용물을 정렬하는 법을 정한다.

em은 부모의 사이즈에 곱한다. 기본 브라우저의 사이즈는 16px이다. 그렇기에 8em이면 16*8 = 128px이다 만약 여기 자식에서 0.5em이면 128*0.5=64px이 된다. 근데 여기서 0.5rem은 root요소에 의해 결정되므로 root요소는 16px이므로 16*0.5=8px이다. 요소가 어떤 곳에서도 동일한 크기로 사용되면 rem , 아니면 em 을 사용하는 것이 좋다.
