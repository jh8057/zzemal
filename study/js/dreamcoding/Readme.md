# undefine

```
(result) => result && alert(`${result.name} has ${result.role} access`)
```

- 이 처럼 작성하면, 만약 result가 undefine일 경우, false이므로 뒤에 함수가 동작하지 않는다.
- 빈 배열이나 object는 false가 아니다.
