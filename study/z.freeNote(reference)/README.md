## 단축키 & 소소한 TIP

- ctrl + t : 새 탭 열기 / ctrl + n : 새 창 열기 / ctrl + w : 창 닫기
- ctrl + L : console창 지우기
- ctrl + ` : vscode termial 닫기
- console 에서 shift + enter 하면 여러줄 타입 가능
- Ctrl + Shift + L : 동일 단어 동시 수정
