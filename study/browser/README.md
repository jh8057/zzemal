# console

- 브라우저를 키고 오른쪽 클릭 > 검사를 누르면 console 창을 볼 수 있다.(F12) Elements > esc눌르면 밑에 나오기도 함.
- 콘솔창에서 자바스크립트를 실행할 수 있다.
- 띄어 있는 웹 페이지를 기반으로 실행이 된다.
- 코드 속에 debbug라는 것을 넣으면 디버깅도 가능하다.

# npm

- npm init
- package.json 생성됨
- npm run `scripts 내 요소` script가 실행됨
